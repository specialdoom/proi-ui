| Property   | Description                            | Type      | Default |
|------------|----------------------------------------|-----------|---------|

| `disabled` | Whether the Checkbox is disabled       | `boolean` | `false` |
| `error`    | Whether the Checkbox is in error state | `boolean` | `false` |
